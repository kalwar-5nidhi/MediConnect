# MediConnect
Healthcare and ePharmacy Web Application
## Overview
MediConnect is a web-based healthcare application built using PHP and MySQL. It provides a platform for users to search healthcare services, access pharmacy-related features, and communicate with the system.
The project focuses on improving accessibility and organizing healthcare services in a simple and structured way.

## Features
- Search healthcare services
- View medical-related information
- Basic ePharmacy functionality
- Emergency service support (extendable)
- User authentication (login and registration)
- Password reset via email
- Contact system to send emails directly
- Modular PHP structure
- Testing using PHPUnit

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Dependency Management: Composer
- Testing: PHPUnit
